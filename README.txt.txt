概览
本数据集包含来自国内某几个教学医院麻醉科住院医师规范化培训过程中的临床操作反馈文本。每条反馈均由带教教师撰写，并已针对其“质量”进行人工标注与多维特征解析，适用于麻醉科医疗教育评估、自然语言处理（NLP）、文本分类模型训练等相关研究。

数据集详情

1. 数据来源与背景
来源:训练测试集数据来自宁波大学附属第一医院麻醉科带教老师的叙述性反馈数据，提取自医院教学管理系统，包括来自三种评估工具的自由文本：住院医师月绩效考核表、迷你临床评估演练量表（Mini-CEX）和操作技能直接观察评估量表（DOPS），经过人工审查排除非麻醉科带教老师的反馈。外部测试选择宁波市李惠利医院麻醉科，数据提取自临床频道（CCMTV）系统中360评价、日常考核、出科考核中的自由文本。
时间跨度: 2024年1月 - 2025年12月
伦理说明:本研究获得了伦理委员会的批准，委员会准予免审并豁免了知情同意要求（批件号：宁波大学附属第一医院伦审2025研第321A号-01） 。所有数据在使用前均已进行严格的匿名化处理，移除了评价教师、住院医师的个人身份信息。

 2. 文件内容
本数据集包含一个主文件：
训练测试集数据.xlsx: 包含 990条记录，每条记录对应一次独立评价。
外部测试数据集.xlsx: 包含 587条记录，每条记录对应一次独立评价。
A列（序号）：记录的唯一编号，从1开始。
B列（评价老师）与C列（住培医师）：分别为带教教师和接受评价的住院医师的匿名化标识，均以“***”代替以保护隐私。
D列（反馈文本）：教师撰写的原始自由文本反馈，内容主要围绕特定临床操作（如气管插管、动脉穿刺、神经阻滞等）的熟练度、操作规范、人文关怀及可改进之处进行描述。该列存在显著的文本缺失情况，有大量记录为空，这反映了在实际评估中反馈填写不完整的普遍现象，是数据集的一个重要特征。
E列至J列（特征1至特征6）：基于共识框架对“反馈文本”进行人工解析后的结构化特征标注。每个特征均为二元变量，取值为0或1（无量纲），分别代表该条反馈是否具备以下质量维度：以住培医师行为为中心的、提供详细信息的、负面反馈、评价专业性和沟通能力的、有针对性的、指出可改进的。
K列（人工标注反馈质量）：基于上述六个特征的综合判断，经过3位专家最终裁定，对每条反馈给出的最终质量分类标签，分为“高质量”与“低质量”两类。
L列（质量评分）：通过Excel公式（=SUM(E2:J2)等）计算得出的定量分数，即六个特征值的加和（范围0-6），为“人工标注反馈质量”提供连续的量化依据。

3. 数据字段说明 (列标签)
| 列名 | 描述 | 取值/格式 |

| 序号 | 记录的唯一标识符 | 整数，1-1000 |
| 评价老师 | 带教教师匿名化标识 |  (已脱敏) |
| 住培医师 | 住院医师匿名化标识 |  (已脱敏) |
| 反馈文本 | 教师撰写的原始自由文本 | 字符串。注意: 存在大量空值，这是数据集的真实特征。 |
| 特征1: 以住培医师行为为中心的 | 二元特征 | 1 (是) / 0 (否) |
| 特征2: 提供详细信息的 | 二元特征 | 1 (是) / 0 (否) |
| 特征3: 负面反馈 | 二元特征 | 1 (是) / 0 (否) |
| 特征4: 评价专业性和沟通能力的| 二元特征 | 1 (是) / 0 (否) |
| 特征5: 有针对性的 | 二元特征 | 1 (是) / 0 (否) |
| 特征6: 指出可改进的 | 二元特征 | 1 (是) / 0 (否) |
| 人工标注反馈质量| 基于上述特征的综合质量标签 | “高质量” / “低质量” |
| 质量评分 | 六个特征值的总和 | 整数，范围 0-6 |

4. 使用建议与潜在应用
核心任务: 反馈文本的自动质量分类。
研究方向: 
     医学教育中的反馈机制分析。
     基于预训练模型（如BERT）的临床文本分类。
    特征工程与解释性AI (XAI) 研究。
注意事项: 使用前请充分考虑“反馈文本”列的缺失值问题，并选择合适的数据清洗策略。
